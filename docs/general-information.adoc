= General information
:chapter-number: 0

[[module-artifact]]
== Artifact

```xml
    <dependencies>
        <dependency>
            <groupId>com.foreach.across.modules</groupId>
            <artifactId>entity-module</artifactId>
            <version>{entity-module-version}</version>
        </dependency>
    </dependencies>
```

[[module-dependencies]]
== Module dependencies

[cols="1,1,3",options="header"]
|===

| Module
| Type
| Description

| AdminWebModule
| optional
| Enables auto generated forms for managing the registered entities.

| BootstrapUiModule
| optional
| Activates support for default Bootstrap based view elements.

|===

[[module-settings]]
== Module settings

EntityModule supports the following configuration properties:
[source,properties]
----
# Customize the message code prefix that should be used for entity messages
# for all entities from that module
entityModule.message-codes[MODULE_NAME] = PREFIX
----
